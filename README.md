# mmWave Channel Model — 40–60 GHz Urban Backhaul

Physics-based fixed terrestrial radio channel model implementing 
ITU-R propagation recommendations from first principles.

## What it does

- Rain attenuation (ITU-R P.838-3)
- Free-space path loss (ITU-R P.525-5)  
- Antenna gain pattern (ITU-R F.699-9)
- Full link budget: EIRP, RSL, fade margin

## In progress
LSTM-based RSL prediction for proactive fade management
Atmospheric gas attenuation (ITU-R P.676-13, 79 spectral lines)
